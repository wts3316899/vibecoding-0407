**ZenFocus** 是一個專為現代數位工作者設計的護眼放鬆網頁。透過「遠近交替對焦任務」，引導使用者規律地切換焦點，幫助緩解長時間盯著螢幕所造成的睫狀肌緊繃與數位眼部疲勞。

## 🌟 核心特色

- **🧘 禪意設計**：簡約、美觀的玻璃擬態（Glassmorphism）介面，營造放鬆的練習氛圍。
- **👁️ 遠近對焦任務**：每 5 秒自動切換一次焦點（近景與遠景），訓練眼部肌肉收縮與放鬆。
- **🌲 動態背景**：內建多款高畫質、具備深度感的自然風景圖，讓你彷彿置身森林或湖畔。
- **🎵 環境音效**：內建放鬆的背景音樂，提升沉浸感。
- **📱 行動優化 & PWA**：完全支援手機瀏覽，並可作為 PWA 應用程式安裝至桌面。

## 🚀 快速開始

你可以直接開啟 `index.html` 即可開始練習，無需安裝任何依賴。

### 練習方法：
1. 點擊 **「開始練習」**。
2. **看近處**：當畫面中心出現圖示（葉子、蓮花等）時，請專注觀察圖示的細節。
3. **看遠方**：當畫面模糊、背景變清晰時，請試著看穿過螢幕，看向背景圖的最深處。
4. 每 5 秒會自動切換一次。

## 📲 安裝至手機案頭 (Android/iOS)

本專案支援 PWA 技術，你可以將它像 App 一樣放在桌面：

1. 使用 Chrome (Android) 或 Safari (iOS) 打開網頁。
2. Android：點擊選單中的 **「新增至主畫面」**。
3. iOS：點擊分享按鈕並選擇 **「加入主畫面」**。
4. 桌面將會顯示專屬的 ZenFocus 圖示。

## 🛠️ 技術棧

- **HTML5**: 語義化結構與音訊處理。
- **Vanilla CSS**: 原生 CSS 變數、Flexbox/Grid 佈局、Glassmorphism 視覺效果。
- **JavaScript (ES6+)**: 核心邏輯控制、動畫與非同步圖片載入。
- **SVG**: 用於輕量化圖示與對焦目標。
- **Web App Manifest**: 支援 PWA 安裝功能。
